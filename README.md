# Skillpool V1

A modern, student-focused Next.js App Router front-end scaffold for **Skillpool** — a student-only skill-sharing marketplace and community hub for STI College Carmona.

## Stack
- Next.js App Router
- Tailwind CSS
- JavaScript
- Supabase-ready structure

## Included pages
- Home
- About Us
- Join Us / Login
- Forum / Listings
- Thread / Post Details
- Programs
- Leaderboards
- Profile

## Notes
- Uses realistic mock data first
- Avoids advanced backend logic in V1
- Includes a simple Supabase schema and integration notes

## Suggested next step
Wire auth first, then replace mock forum data with real Supabase queries.
